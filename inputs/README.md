Bundled inputs for the Nunez-Elizalde 2022 converter.

This folder stores `Protocol.mat` files required to reconstruct BIDS stimulus
`events.tsv` files from the public `Subjects` release, which does not include these
MPEP protocol definitions.

The files under `protocols/<subject>/<date>/<expID>/Protocol.mat` mirror the original
session/block layout and were copied from the corresponding Cortexlab acquisition
storage.
